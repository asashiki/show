特性

内联渲染

基于 MCP Apps 组件，在 claude.ai 与 ChatGPT 网页端直接显示，不只是贴链接。

按心情挑图

内置情绪 / 场景匹配，AI 描述一下就能选中合适的表情。

图库可成长

AI 能通过临时上传链接或现有图片 URL 往库里添加新表情。

网页后台

/admin 支持拖拽上传、批量操作、打标签、搜索与删除。

轻量存储

JSON 文件 + 磁盘图片，无需数据库，自托管友好。

可选鉴权

MCP_AUTH_PASSWORD / ADMIN_TOKEN 保护接口与后台。

快速开始

# 构建并启动（Streamable HTTP，默认 :3000）
npm install
npm run build
npm start

# Claude Desktop（stdio）
npm run start:stdio

Claude Desktop 配置：

{
  "mcpServers": {
    "sticker": {
      "command": "node",
      "args": ["path/to/sticker-mcp/dist/stdio.js"]
    }
  }
}
